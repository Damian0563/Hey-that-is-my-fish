# epfu24W-103-D-penguins

This project was made to house the Penguin board game for Programming fundamental class. The project is done and managed by group D, consisting of: Nguyen Huynh Gia Bao, Priyanshu Rajput and Damian Piechocki.

To compile the code, please use the following command in your terminal:

    gcc main.c movement.c placement.c structs.c file_handling.c utils.c

The compiler runs with no errors, there is also a unit test file that we use to check the functions if it works or not, as it is for the debugging and unit testing process, it is not added here.

Run file with commands given in penguins_v2 pdf file. 

html folder houses the documentation of the penguin game app. Please choose index.html within the html folder when you check as it is the main page of the documentation.
