This function moves the penguin autonomously using logic.

Moves the penguin autonomously using logic.

This function iterates through the 2D array to find the best move in the current game state by checking the fish on and around the floes. It checks in 4 directions for the best move, then checks again around that new best move location to find potentially more fish, if the total comes back as the best move, the penguin is moved.
